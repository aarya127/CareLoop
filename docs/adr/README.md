# Architecture Decision Records

ADRs document significant technical decisions.

## Format

Each ADR lives in its own file: `adr/NNNN-short-title.md`

```markdown
# ADR-NNNN: Title

## Status
Accepted | Deprecated | Superseded by ADR-XXXX

## Context
What problem were we solving?

## Decision
What did we decide?

## Consequences
What are the trade-offs?
```
